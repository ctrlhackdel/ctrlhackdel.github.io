# Tech Stack
- **Framework:** Next.js w/ App Router
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **Database:** Postgres (Hosted using NeonDB)
- **ORM:** DrizzleORM
- **Authentication:** BetterAuth
- **Icons:** Lucide Icons
- **Form Handling:** React Hook Form + Zod
- **Storage:** Amazon Simple Storage Service (S3)
- **Emails:** Amazon Simple Email Service + Nodemailer (More on this later)
- **Error Logging and Capture:** Sentry

# Project Structures

Each iteration we experimented with a different structure that allowed us to experiment with organization and different routing of the websites.
